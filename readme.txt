	Phantom Bullet Dimension.


	～弾幕STGの一つの結論～




・動作環境
Windows 64bit
DirectX9以降が使用可能



・操作方法
移動		:十字キー
ショット,決定	:Z
低速移動,戻る	:Shift
ポーズ		:Escape
*config.iniでキーコンフィグができます。



・設定の変更
config.iniというファイルを開いて各項目を設定してください。
項目の順番を入れ替えると正常に動作しないので注意。

-各項目の解説-

[Screen]
	主に画面の設定を担当します。
WindowMode
	1にするとウィンドウで、0だとフルスクリーンで起動します。
	フルスクリーンの場合はウィンドウ関係の項目を無視します。
X,Y
	ディスプレイの左上を原点としたウィンドウのXY座標です。
Width,Height
	ウィンドウのサイズをピクセル単位で変更します。
	このゲームは640x480の解像度で動作しているため、
	WidthとHeightを変更する場合は比率を4:3にしておくべきです。

[Process]
	内部処理の設定です。
	この辺は下手に変えないほうがいいと思います。
VSync
	垂直同期を設定します。1で有効、0で無効。
SleepType
	ウェイトの処理を設定します。1で独自の高精度処理、0で通常の処理。
BltPrepareTime
	画面に描画する前に処理を開始する時間の余裕をミリ秒単位で指定します。
	0未満と16以上は指定できません。
	動作が重い時は数字を増やすと安定するかもしれません。
GraphicalEffect
	ゲームプレイに関係ない画像エフェクトや背景を表示します。1で有効、0で無効。
DrawMode
	描画アルゴリズムを設定します。0でニアレストネイバー法、1でバイリニア法。
	ニアレストネイバー法は処理が早いですが、描画した時にドットが荒くなります。
	バイリニア法は綺麗に描画されますが、処理が遅いです。

[KeyConfig]
	入力に関する設定です。
Shot,Slow,Pause
	それぞれの操作のキーコンフィグができます。
	キーボードの場合は以下の通り
	1:Z  2:X  3:C  4:A  5:S  6:D  7:Q  8:W
	9:Escape  10:Space  11:Shift  12:Control
JoypadDeadZone
	パッドのスティックの傾きを無視する範囲です。0から100で設定。

[Option]
	主にゲームプレイに関する設定です。
	これらの項目はゲーム内でも変更できます。
MusicVolume,SEVolume
	音楽と効果音の音量を調整します。0から100で設定。
MoveSpeedFast,MoveSpeedSlow
	それぞれ高速移動と低速移動の速度を設定します。
	(設定値/4.0)pxが1フレームに移動する量です。



・リプレイについて
ゲームオーバー又はクリアした時にSave Replayを選択すると保存できます。
replayフォルダの中に「rp（数字二桁）」という名前で保存されます。（例：rp00,rp13）
リプレイ閲覧時はrp00～rp15までしか読み込まれませんが、保存はrp99まで可能です。



・設定の初期化
config.iniを削除してください。
ゲーム内のOption項目でSaveを選択すると設定ファイルを再生成できます。



・アンインストール方法
ゲームが入っているフォルダごと削除してください。
レジストリや個人ディレクトリは使ってないです。



・Q&A

Q:○○.dllというファイルが存在しませんと言われる。
A:DirectX Runtimeをインストールしましょう。

Q:Microsoft Difender Smartscreenにブロックされる。
A:このゲームは電子署名を行ってないので表示されます。ググって解除してください。

Q:バグらしき挙動を発見した。
A:ソースコードは公開されてあるので自力で直してください。
